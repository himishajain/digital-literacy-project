##  Project Overview
This project was created as part of the Digital Literacy course (CSE0001).  
The aim of this project is to understand and apply essential digital skills such as creating an online presence, using digital tools, communicating professionally, and staying safe online.

The project is divided into five tasks, each focusing on a different aspect of digital literacy.

## Project Structure

### 🔹 Task 1 – Infographic
Created a digital literacy infographic using Canva to explain key concepts like safe internet practices and professional presence.

### 🔹 Task 2 – Digital Portfolio
Created professional profiles on platforms like GitHub LinkedIn and kaggle to build an online presence.
